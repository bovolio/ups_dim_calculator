# Dimensional Weight Calculator
--------------------
Dimensional Weight Calculator for UPS and Purolator ground services. made this because UPS' site is god damn slow.

## Requirements
* Web Server (Apache, Nginx, LiteSpeed, Lightly, IIS, etc.)
* PHP 5.6.3 or higher
